# Button

按钮用于开始一个即时操作。

## 何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

## 基本使用

```tsx
import React from 'react';
import { Button } from 'ui-starter'

export default () => {
  return <Button>button</Button>
}
```
